| Aspect             | Acquisition             | Merger                    |
| :----------------- | :---------------------- | :------------------------ |
| **Nature**         | One buys another        | Two companies unite       |
| **Identity**       | Target loses identity   | New identity forms        |
| **Control**        | Acquirer gets control   | Shared control emerges    |
| **Size**           | Often unequal size      | Usually similar size      |
| **Consent**        | Can be hostile          | Mostly friendly agreement |
| **Process**        | Relatively simpler      | More complex process      |
| **Stock**          | Buyer's stock continues | New stock issued          |
| **Name**           | Acquirer's name remains | Often new name            |
| **Power Dynamics** | Clear power hierarchy   | Balanced power structure  |
| **Example**        | WhatsApp by Facebook    | Disney-Pixar merger       |
